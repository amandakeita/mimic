definition: a program which captures frames from the webcam of a linux machine, turns those jpeg frames into managable of processable pixels which then get interpreted as coordinates which in turn return the closest match from manually currated set of coordinates alligned with emotions. The jpeg decompressor much like everything else is manually made. the code aims to avoid pre-made structures to maximalise learning.
